# PBRLighting

A demonstration of implementing PBR lighting. The implementation is based on Cook-Torrance BRDF under the concept of reflectance equation which is a simplified version of rendering equation. It shows some sort of realism in terms of lighting.

The video link is at https://youtu.be/7KkZrb9Bem8.

To use this program, press up and down arrow keys to change roughness of the sphere, and press left and right arrow keys to change metallic of the sphere.
